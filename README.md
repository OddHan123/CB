# CodeBook-Sample
> An example for codebook

## Info

### codebook.tex
> base information (ex. team name)
```tex
\fancyhead[C]{標題一}
\fancyhead[L]{標題二}
```

### content.tex
> contents (ex. codes, theorems)
```tex
\section{Section1}
    \subsection{Subsection1}
        \lstinputlisting{route/to/your/code}
        \input{route/to/your/theorem.tex}
```
